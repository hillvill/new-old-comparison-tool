# 新旧対照表作成ツール

法務・総務担当者のための、ブラウザで完結する新旧対照表作成ツールです。
Wordファイル（.docx）を2つ読み込ませるだけで、精度の高い新旧対照表を自動生成します。

👉 **[今すぐツールを使う（ブラウザで起動）](https://[あなたのユーザー名].github.io/[リポジトリ名]/)**

## 💡 特徴 (Features)

* **完全ローカル処理（セキュリティ対応）**
  すべてのデータ処理はご利用のPC（ブラウザ内）で完結します。
* **マッピング（事前調整）機能**
  全自動による「意図しないズレ」を防ぐため、比較前に新旧の条文紐付けを確認・修正できる「マッピング画面」を搭載しています。
* **表記ゆれの吸収**
  「及び・および」「1・１」といった実務上影響のない表記ゆれや、単純な条番号のズレを「維持（軽微修正）」として処理し、レビューのノイズを減らします。
* **コピー＆ペーストで完結**
  出力された表は、ボタン一つでクリップボードにコピー可能。普段お使いのExcelやWordに貼り付けて、自由にレイアウトを整えられます。

## 🚀 使い方 (Usage)

特別なソフトのインストールや会員登録は不要です。

1. **ファイルの選択 (STEP 1)**
   比較したい旧版と新版のWordファイル（.docx）を選択し、「条文を解析」をクリックします。
2. **マッピングの確認 (STEP 2)**
   システムが自動で新旧の条文を紐付けます。条文の新設・削除などでズレが生じている場合は、ドロップダウンリストから正しい組み合わせを選択して修正します。
3. **対照表の生成 (STEP 3)**
   「対照表を作成する」をクリックすると、追加変更が赤字、削除が下線で示された対照表が生成されます。
4. **コピーして活用**
   「表をコピーする」ボタンを押し、ExcelやWordに貼り付けてご利用ください。

## 🛠 使用技術・ライブラリ (Tech Stack)

高度な解析のために、以下のオープンソースライブラリを使用しています。

* [mammoth.js](https://github.com/mwilliamson/mammoth.js) (Word構造解析)
* [jsdiff](https://github.com/kpdecker/jsdiff) (差分抽出エンジン)
